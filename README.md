# Geulgyeol (Automated Essay Scoring Agent)

## Project Introduction
Geulgyeol is an open-source tool designed to streamline the repetitive task of evaluating and scoring Korean essays. By leveraging an AI agent, it organizes complex grading tasks into a structured, easy-to-read format.

## Reason for Making
The process of manually reviewing information (essays) and formatting the feedback into a consistent structure is cumbersome and time-consuming. We started this project because we wanted to create a system where users can check reliable, formatted results with just simple inputs, eliminating the repetitive manual work of essay evaluation.

## Main Features
- Organizes input content (essays) into a predefined structure based on 8 scoring rubrics.
- Allows users to check the final output in the form of a detailed file or document report (including overall reviews, strengths, and areas for improvement).
- Provides example files and criteria so users can easily understand how to use the tool.
- Provides a basic workflow that significantly reduces repetitive tasks related to grading.

## How to Use
1. Check the example files and scoring criteria provided.
2. Input the necessary content (the essay) to be evaluated.
3. Run the application and check the generated evaluation results.
4. Modify the output format or rubric weights as needed.
